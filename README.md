# Posting Page

Projeto de exemplo para simular a criação de um post usando `fetch` e a API pública jsonplaceholder.

Como executar

1. Abra o projeto na pasta `posting--page` no VS Code.
2. Abra `index.html` com Live Server (extensão Live Server) ou abra o arquivo diretamente no navegador.
3. Preencha o `Título` e `Conteúdo` e clique em `Publicar` — o post será enviado para `https://jsonplaceholder.typicode.com/posts`.

O que está neste repositório

- `index.html` — formulário e área de renderização do post
- `style.css` — estilos básicos
- `main.js` — lógica JavaScript: seletores, evento `submit` e `fetch` POST

Observações

- A API usada é apenas para testes (jsonplaceholder) e retorna sempre um ID simulado.
- Não há persistência real neste exemplo (apenas demonstração do fluxo frontend → API).
